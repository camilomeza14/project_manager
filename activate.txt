*Activar repo* 
En el computador haces esto:
1. Clona el repositorio:
bash git clone https://github.com/camilomeza14/project_manager

2. Entra a la carpeta:
 cd project_manager

3. Instala json-server:
 npm install -g json-server

4. Arranca la API:
 json-server --watch db.json --port 3000

5. Abre index.html con Live Server desde VS Code.

Lo único que necesitas tener instalado en ese computador es Node.js y VS Code con Live Server. Git casi seguro ya está.


**Verifica primero instalacion de node.js:**

node -v
Si aparece un número como v18.x.x → ya está, no necesitas instalar nada.



**Si no lo tiene, sigue según el sistema operativo del computador**:

*en computadores LINUX*
sudo apt update
sudo apt install nodejs npm
node -v

En todos los casos al final verifica con node -v que muestre un número. Si aparece → listo, puedes correr el proyecto.
